# Glossary

## User

A person who uses the system.

## Organization

A workspace that owns users, watchlists, events and alerts.

## Watchlist

A list of objects that should be monitored.

## Watchlist Item

A specific monitored object, for example:

- email
- phone
- vehicle_id
- user_id
- card_id
- ip_address

## Event

An incoming fact from an external system.

## Alert

A system-generated record that requires user attention.

## Audit Log

A history of important user and system actions.
